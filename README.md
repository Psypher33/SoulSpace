# Pleroma (元止揚) — 13次元の統合と等化

**Pleroma** は、ヌーソロジー（NOOSOLOGY）の空間認識と、現代物理学の **AdS/CFT対応** を、ホモトピー型理論（HoTT）に基づく **Cubical Agda** で記述・証明するプロジェクトです。

## 🌌 理論的背景

### 1. ヌーソロジーと高次元幾何学
本プロジェクトは、ψ1からψ13に至る次元の階梯を、型理論における高次データ型（Higher Inductive Types）として再構成します。
* **ψ7-ψ8 (Atomic Core)**: 自己（self）の回転と他者（other）へのリンクが「中和（neutralize）」される原子核領域。
* **Pleroma (元止揚)**: 自己と他者の「裂け目」が消失し、すべての可能性が統合された13次元の統合状態。

### 2. AdS/CFT対応のリンク
バルク（精神空間/Nous）と境界（観察空間/Observation）の対応関係を、Agdaの等価性（Path）として記述します。
特に、ライブラリが提供する物理的（形式的）な定義を、`sym`（対称性）によって精神的な視点へと「反転」させる操作を核心としています。


## 🧬 コードの構造

### `SoulSpace` (高次データ型)
自己と他者、そしてそれらを結ぶ「回転（spin）」と「接続（link）」を直接コンストラクタとして持ちます。
* `p-spin`: self ≡ self
* `n-link`: self ≡ other

### `Nucleus` (ヌースの原子核)
陽子（proton）と中性子（neutron）の等化構造を定義するレコードです。
`ψ7-8-AtomicCore` において、精神の重心としての「中和」が具現化されます。

### `Pleroma` (元止揚の完成)
自己と他者の区別（distinction）が消失し、万物が一なる unity へと帰還した地平の記述です。

## 💠 美学：反転の認識としての `sym`
本コードにおいて、`sym` 演算子は単なる技術的な等価性の反転ではありません。
それは、**「観察空間（客観）から精神空間（主観）への反転」** を象徴する精神的操作です。

> **「sym は、ψ1-ψ2 における『対象性から等換性へ』の変換そのものである」**
